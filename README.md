# Python Tutorial

This is the material used for the Python tutorial for the HSF/IRIS-HEP Software Basics Training at BNL (June 2026). The material is taken from the following existing material.

- Sections 1.* are taken from the [HSF Training Analysis Essentials module](https://hsf-training.github.io/analysis-essentials/index.html)
- Sections 3.0 and 3.1 are taken from [Level Up Your Python](https://henryiii.github.io/level-up-your-python/)
- Section 3.2 is taken from the [HSF Training Array-Oriented programming module](https://hsf-training.github.io/array-oriented-programming/0-intro.html)
- Sections 4.* are taken from the [HSF Training Scikit-HEP module](https://hsf-training.github.io/hsf-training-scikit-hep-webpage/)
